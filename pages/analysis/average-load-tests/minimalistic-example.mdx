# Minimalbeispiel

Im Minimalbeispiel zeigt sich ebenfalls eine ausgeprägte Streuung der Antwortzeiten der
virtuellen Threads, wenngleich diese geringer ist. Dies ist in erster Linie auf den kleineren
Wertebereich zurückzuführen. Die Antwortzeiten tendieren dazu, näher beieinander zu liegen,
was auf die insgesamt kürzeren Antwortzeiten im Zusammenhang mit der geringeren Komplexität
des Beispiels zurückzuführen ist. Die Beobachtung dieser Streuung ermöglicht eine präzisere
Analyse der Leistung der virtuellen Threads in einem weniger komplexen Szenario und trägt dazu
bei, etwaige Muster oder Unregelmäßigkeiten zu identifizieren.
