# Reaktive Programmierung

## Project Reactor

Die reaktive Entwicklung in Java mit Project Reactor basiert auf dem Konzept der reaktiven Programmierung, das darauf abzielt, asynchrone und nicht blockierende Operationen effizient zu verarbeiten. Reactor ist eine Open Source Bibliothek für reaktive Programmierung in Java, die auf dem Reactive Streams-Standard aufbaut.
Das Herzstück der reaktiven Programmierung ist die Event Loop, auch als "Reaktor" bezeichnet. Die Event Loop ist ein Mechanismus, der kontinuierlich auf Ereignisse wartet und diese effizient verarbeitet, ohne den Hauptthread zu blockieren. In Reactor wird die Event Loop verwendet, um asynchrone Operationen zu steuern.
